安装完毕后，打开 cmd，使用 nvm install 指令安装指定版本的 nodejs

然后用 nvm list 查看是否安装成功

看到版本号前的星号就表示安装成功，并且当前使用该版本

nvm 工具管理 nodejs 版本非常方便，其他指令可自行查询
